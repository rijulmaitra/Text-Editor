## Deployment

1. **Prerequisites**

      #### For Debian/Ubuntu
         sudo apt-get install python3-tk

      #### For Red Hat/Fedora
         sudo dnf install python3-tkinter

      #### For macOS (with Homebrew)
         brew install python-tk
